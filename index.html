  • Coded a single layer neural network from scratch to classify a subset of images in the CIFAR-10 dataset

Named Entity Recognition Project

  • Coded Hidden Markov Models to implement a named entity recognition system
  • Classified named entities as a person, location, or organization
  • Used the English and French subsets of the WikiANN dataset to train
  • The Hidden Markov Model parameters were learned given the training data
  • Then the forward backward algorithm was implemented to perform a smoothing query
  • This was then used to predict the hidden tags for a sequence of words

Reinforcement Learning Project

  • Implemented Q-learning with linear function approximation to solve the mountain car environment
  • The mountain car environment consists of a car with a set amount of power which is in a valley
  • To solve the problem the car must learn how far it must reverse up the hill behind it in order to leverage gravity and power to reach a flag on the hill in front of the car
  • The epsilon-greedy method was used for action selection
